# ATM-Interface

A simple console-based ATM system built using Java that supports balance inquiry, deposit, withdrawal, and exit operations.

## Features

* Check balance
* Deposit and withdraw money
* Input validation & error handling
* Menu-driven console interface

## Concepts Used

* Java OOP (classes, objects, encapsulation)
* Exception handling
* CLI interaction with `Scanner`

## Future Improvements

* PIN authentication
* Transaction history
* GUI support
* Database integration
